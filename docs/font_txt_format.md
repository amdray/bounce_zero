# Формат TXT для шрифтов (человекочитаемый)

Формат рассчитан на ручное редактирование и детерминированную конвертацию в T4‑атласы.
Все кодпоинты задаются как `U+XXXX`, файл остаётся ASCII.

## Заголовок

Обязательные строки (порядок не важен):

```
font name=font23 height=23
range U+0000..U+0451
default U+0020
```

- `height` — высота глифа в пикселях.
- `range` — диапазон атласа. Отсутствующие глифы в диапазоне считаются пустыми.
- `default` — символ по умолчанию, если кодпоинт отсутствует.

## Блоки глифов

```
glyph U+0041 width=10
0011000000
0111100000
...
end
```

Правила:
- Каждый глиф содержит ровно `height` строк битовой маски.
- Используются только `0` и `1` (`1` — пиксель включён).
- `width` задаётся явно (для пробела и корректного advance).
- Длина каждой строки должна совпадать с `width`.

Комментарии:
- Строки, начинающиеся с `#`, игнорируются.

## Примечания
- Новые символы добавляются обычными блоками `glyph`.
- Дубликаты кодпоинтов запрещены (конвертер выдаст ошибку).
